### What is this project ###
android-camera can turn devices from android 11+ that are old or unused and make them web cams

## Installation ##
to install this tool as root run
` shc -f main.sh -o /bin/android-camera-wireless `

and just like that it becomes a command on your system

## Tested devices ##

Unfortunatly I only own one android device so I could only test this for the Samsung Tab A9+

VPN's ruin this version as this is the tcpip version and I do plan to make a wired version for a better camera
